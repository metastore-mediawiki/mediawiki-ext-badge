# Информация

Интеграция значков в статью.

## Синтаксис

```
{{#badge: user|music}}
```

## Установка

1. Загрузите папки и файлы в директорию `extensions/MW_EXT_Badge`.
2. В самый низ файла `LocalSettings.php` добавьте строку:

```
wfLoadExtension( 'MW_EXT_Badge' );
```
